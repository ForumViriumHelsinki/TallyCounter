# Tally Counter
To start UI development:

* Install node.js
* Run:
```
npm install -g yarn
cd react_ui
yarn
yarn start
```
This should install all dependencies in react_ui/node_modules and start a react dev environment serving the UI app at localhost:3000 or a nearby available port.
